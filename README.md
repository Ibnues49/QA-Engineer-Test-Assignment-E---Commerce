Tes terdiri dari dua bagian utama:

**Manual Testing (40%)
Automation Testing menggunakan Katalon Studio (60%)**

Website yang diuji: https://demowebshop.tricentis.com/

**Section 1 – Manual Testing**
terdapat pada file : Testcase E- Commerce_Ibnu Aqil.xlsx

**Section 2 – Automation Testing (Katalon Studio)**
Pada bagian ini saya membuat script automation untuk memvalidasi halaman Product Catalog Page, dengan langkah:

- Membuka browser dan mengunjungi halaman utama
- Memverifikasi bahwa daftar produk tampil
- Mengklik salah satu produk
- Memverifikasi bahwa detail produk tampil dengan benar (nama & harga)

**Cara Menjalankan Test (Katalon Studio)**
1️⃣ Buka Project
- Buka Katalon Studio
- Klik File → Open Project
- Pilih folder repository ini atau bisa didownload

2️⃣ Jalankan Test Case
- Masuk ke folder Test Cases
- Pilih TC_001_Product_Catalog
- Klik Run

3️⃣ Kebutuhan Sistem
- Katalon Studio 9.3.1+
- Web browser (Chrome / Edge)
- WebDriver update dari Katalon
